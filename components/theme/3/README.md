# Theme 3 Documentation - "Netflix Inspired"

This directory contains the source code for Theme 3, a digital invitation theme inspired by the Netflix UI.

## File Structure

```

```

## Component Architecture

The main component `Theme3` in `page.tsx` receives a `data` prop of type `ThemeData`.
It manages the state for:
- Opening animation (Profile selection)
- Navigation (Home, Event, Gallery, etc.)
- Data display (Bride/Groom, Events, Story, etc.)

## Customization Guide

### Helper Functions & Data
The theme relies on `items` from `lib/theme-data.ts`.
Key interfaces:
- `ThemeData`: Main data structure containing all invitation details.
- `EventData`: Structure for event details (Date, Time, Location).

### Dimensions & Responsive Design
- **Mobile First**: The design is optimized for mobile screens (portrait).
- **Desktop**: Centered container with max-width (usually `md` or `lg` tailwind classes) to simulate a mobile app experience.

### Adding New Sections
1. Create a new `<NetflixSection>` in `page.tsx`.
2. Use `<NetflixHeader>` for titles.
3. Use `<NetflixText>` for content.

### Styling
- Uses Tailwind CSS.
- **Colors**:
  - Background: Black (`bg-black`)
  - Accent: Red (`text-red-600`)
  - Text: White/Gray

## How to Duplicate for New Theme

1. **Copy Folder**: Copy `components/theme/3` to `components/theme/4`.
2. **Rename Component**: inside `page.tsx`, rename `function Theme3` to `function Theme4`.
3. **Update Imports**: Ensure imports point to the correct local components or shared libraries.
4. **Modify Design**:
   - Change `NetflixComponents` to your new design system.
   - Update layout in `page.tsx`.
   - Replace icons in `icon/`.
