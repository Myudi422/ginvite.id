# Theme 4 Documentation - "Exclusive Video"

This directory contains the source code for Theme 4, a premium digital invitation theme featuring a full-screen video opening and an exclusive, elegant design.

## Key Features

-   **Full Screen Video Opening**: Automatically plays a video background (muted, looped) in the opening section if `content.plugin.opening_video` or `content.plugin.youtube_link` is provided.
-   **Exclusive Design**: Gold, Black, and White color palette with Serif typography.
-   **Vertical Layout**: Clean, scrolling layout replacing the carousel-based design of Theme 3.
-   **Glassmorphism**: Usage of backdrop-blur and semi-transparent backgrounds for a modern premium feel.

## Customization Guide

### Video Background
To enable the video background in the opening:
1.  Set `content.plugin.opening_video` to a direct URL of an MP4/WebM file (Recommended).
2.  Alternatively, `content.plugin.youtube_link` will be used if `opening_video` is missing (Embeds YouTube player).
3.  Fallback: If neither is present, it uses the first image from `gallery` or `defaultBgImage`.

### Styling
-   **Colors**: defined in `ThemeComponents.tsx` (using Tailwind classes like `text-amber-500`, `bg-zinc-950`).
-   **Fonts**: Uses system serif/sans-serif. To use custom fonts, ensure they are loaded in `layout.tsx` or `globals.css` and referenced in `ThemeComponents.tsx`.

### Adding New Sections
1.  Import `ThemeSection`, `ThemeHeader`, `ThemeText` from `./ThemeComponents`.
2.  Wrap content in `<ThemeSection>`.
3.  Use `<ThemeHeader>` for section titles (uppercase, gold).

## Differences from Theme 3
-   removed `Netflix` specific components and branding.
-   removed Horizontal scrolling carousels.
-   Added `<video>` tag support for opening.
